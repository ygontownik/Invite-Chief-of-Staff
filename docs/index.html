Choose your path

Two different setups depending on whether you're joining an existing team or starting fresh.

Joining a team
Your firm is already set up
Someone at your firm has already configured the pipeline. You just need to get your own machine running and connected.
  • 20 minutes total
  • No Google Cloud setup required
  • Config repo already exists
  • Shared Google Docs already created
→
New firm
Starting from scratch
You're the first person at your firm to set this up. You'll create the accounts, config, and shared docs — then add teammates later.
  • ~60 minutes total
  • Anthropic + Google Cloud setup
  • Creates your shared Google Docs
  • Adds teammates in minutes after
